# Noisecore

Noisecore is a responsive audio e-commerce website developed as a Human-Computer Interaction (HCI) project. The platform provides an intuitive shopping experience for users looking for headphones, earbuds, and speakers.

## Features
- Product catalog browsing
- Category filtering
- Promotional deals section
- About Us page
- Membership registration
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Pages
- Home
- Products
- About
- Deals
- Join Membership

## Purpose
This project was created to apply Human-Computer Interaction (HCI) principles, focusing on usability, accessibility, and user-centered design.
